### Capstone Project 1 - Body Metrics Analysis (NHANES Dataset)
 
An exploratory data analysis project using the **NHANES 2020 Adult Body Measurements** dataset, comparing physical health metrics between males and females.
 
**Key Steps:**
- Loaded and previewed male & female CSV datasets
- Visualized weight distributions using histograms and boxplots
- Computed descriptive statistics — mean, median, std dev, skewness, kurtosis
- Calculated **BMI**, **Waist-to-Height Ratio (WHtR)** and **Waist-to-Hip Ratio (WHR)**
- Applied **Z-score standardization** for consistent scaling
- Generated **scatterplot matrices** to explore pairwise correlations
- Compared lowest vs. highest BMI groups
- Discussed advantages and limitations of BMI vs. ratio-based health metrics
 
**Tools & Libraries:** pandas, numpy, matplotlib, seaborn, scipy.stats
 
**Dataset:** NHANES Adult Body Measurements 2020 — CDC
